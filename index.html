Landmark Tool

Generic image coordinate calibrator

Image

Edge Snap

Edge map: not computed

Sobel edge detection finds tissue boundaries. New landmarks auto-snap to the nearest edge within the radius. Status updates in real time.

Layout Preset

For first-time use of a new image, place landmarks by hand then "Save current as preset" — next time, just pick your preset and drag any misaligned points. Coordinates are stored as fractions of image size, so presets adapt to any image dimensions.

Calibration

Click "Set scale", then click two points a known real-world distance apart and enter the distance in cm.

Mode

In Edit mode, drag points. In Preview, points are tiny dots (final rendering view).

    Connection Style

    Smooth curve — wall/boundary tissue (passes through every point)
    Straight line — flow tract, annulus, or rigid structure
    Free point — no connection, just a reference marker

    Click a landmark in the list to see its distance to connected neighbors. Calibration is needed for cm values.

    Export

    JSON = coordinates only (for simulator). SVG = portable image+landmarks file.

    Help

    Scroll → zoom in/out

    Alt+drag → pan

    Click point in list → select & highlight

    Del on selected → remove landmark
